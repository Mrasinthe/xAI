# xAI Service

## Installation instructions (python 3.8.10)

- git clone https://github.com/Mrasinthe/xAI.git
- cd xAI


## Run the server

- python3 -m venv venv
- source venv/bin/activate
- pip install -r requirements.txt
- python3 main.py

## Deploy using Docker

- Go to xai_docker folder and run 'sudo docker-compose up --build'
